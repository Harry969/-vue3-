# 🌟 Cursor_Blog

使用Cursor基于TS的个人博客系统与自动化工具集。

## ✨ 主要功能

- 📝 博客系统

  - 支持 Markdown 写作
  - 周记管理与自动生成
  - 内容自动化发布

- 🤖 自动化工具

  - ⏰ 自动打卡功能
  - 📊 数据统计分析
  - 🔔 智能提醒系统

- 🌍 多语言支持

## 🚀 快速开始

# 克隆项目

git clone git@github.com:Harry969/Cursor_Blog.git

# 安装依赖

npm install

# 启动开发服务器

npm run dev
